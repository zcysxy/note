# R

R is a [[Scripting Language]] for [[Statistics|statistical computing]] and graphics. It is an open-source implementation of the S programming language, and has stronger object-oriented programming facilities.

R has mature integrations with [[Markdown]] and [[LaTeX]].
You can run R commands within a [[LaTeX]] document.
Also, R is supported by [[Jupyter]] by default.
