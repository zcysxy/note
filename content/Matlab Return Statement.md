# Return Statement

[[Matlab Control Statements]]

---

程序终止语句, 即 _return_ 语句, 是 MATLAB 四大**程序结构控制语句**之一, 作用为字面意义: 终止程序运行, 不再执行 _return_ 语句之后的代码.
