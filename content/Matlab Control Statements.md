# Control Statements

[[MATLAB]]

---

MATLAB 有四大**程序结构控制语句**, 它们由相应关键词和 _end_ 语句表明:

1. 分支控制语句 - _if_, _else_, _elseif_ / _switch_, _case_, _otherwise_
2. 循环控制语句 - _for_ / _while_ + _continue_, _break_
3. 错误控制语句 - _try_, _catch_
4. 程序终止语句 - _return_
