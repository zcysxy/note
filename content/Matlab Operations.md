# Matlab Operations

In MATLAB, "operation" does not refer to mappings in the broad sense; it means simple, direct, at-most-binary calculations expressed via [[Matlab Operator|operators]] or operator-equivalent functions. They fall into four kinds:

- [[Matlab Arithmetic Operations]]
  - [[Matlab Array Operations]]
  - [[Matlab Matrix Operations]]
- [[Matlab Relational Operations]]
- [[Matlab Logical Operations]]
- [[Matlab Set Operations]]

For the first three kinds, this note focuses on **[[Matlab Operator]]** usage. MATLAB has no operator for [[Matlab Set Operations]], so that kind is implemented purely through functions.
