# Matlab Operations

MATLAB 运算 operation 不是指广义的映射, 而是指简单的, 直接的通过[[Matlab Operator|运算符]]和运算函数完成的至多二元的运算. 主要分为以下四类:

- [[Matlab Arithmetic Operations]]
  - [[Matlab Array Operations]]
  - [[Matlab Matrix Operations]]
- [[Matlab Relational Operations]]
- [[Matlab Logical Operations]]
- [[Matlab Set Operations]]

其中前三类, 笔记主要介绍 **[[Matlab Operator]]** 的用法, MATLAB 中没有 [[Matlab Set Operations]] 的运算符, 因此只能通过简单函数实现.
