# Random Variable

Given a [[Sigma Field|measurable space]] $(\Omega,\mathcal{F})$, a random variable is a function $f:\Omega\to\R$ if $\{ \omega: X(\omega)\le c \}$ is $\mathcal{F}$-[[Measurable]] for any $c\in\R$.

We call a r.v. an extended r.v. if the range of $f$ considers $\infty$.

Note that the [[Borel Sigma Field]] can be generated by $\{ (-\infty,c]: c\in\R \}$. Therefore, for any [[Borel Sigma Field|Borel Set]] $B$, $X^{-1}(B) \coloneqq \{ \omega: X(\omega)\in B \}$ is $\mathcal{F}$-measurable.

## Law

Let the measure space be a [[Probability Space]], then a r.v. introduces a ==probability law==:

$$
P_{X}: \mathcal{B}\to [0,1], B\mapsto P_{X}(B)=P(X\in B).
$$

This is also called the ==distribution== of $X$.
Note that a law is different from a [[Cumulative Distribution Function]], but [[Cumulative Distribution Function#^law-cdf|they uniquely determine each other]].

Importantly, $P_{X}$ is a probability [[Measure]]. So $X$ provides a mapping between two probability spaces:

$$
(\Omega,\mathcal{F},P) \overset{ X }{ \longleftrightarrow } (\R,\mathcal{B},P_{X}).
$$

More concretely, the indicator function $1_{A}$ is a r.v. for all $A\in\mathcal{F}$. And a ==simple r.v.== is a linear combination of indicator functions.

## Properties

![[Measurable#^rv]]

- Given a sequence of r.v.s $X_{n}$, the following limits are r.v.s: $\sup_{n}X_n$, $\inf_{n}X_n$, $\limsup_{n}X_n$, $\liminf_{n}X_n$.
